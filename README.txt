Please follow the instruction below the run the project. 

1. Download and install "SoapUI Open Source" from https://www.soapui.org/downloads/soapui.html
2. Open SoapUI-5.4.0
3. Click on “File” and then click on “Import Project”.
4. Select "REST-Project-1-soapui-project" and click on “Open”.
5. Expand “REST Project 1”.
6. Double click on "https://api.github.com TestSuite".
7. Click on “Run” icon.

All Test Cases will be executed and the results will be displayed.
